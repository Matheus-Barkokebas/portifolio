Ferramentas e lingaguens que eu uso

Java

Durante a pandemia, tive a oportunidade de aprender Java por meio de um curso que me ensinou bastante sobre a linguagem. No entanto, com a volta das aulas no ensino médio, precisei interromper o curso devido à falta de tempo. Ainda assim, adquiri um bom conhecimento dos comandos básicos de Java.

Python

Me interessei por Python quando tive o primeiro contato com a linguagem na faculdade e decidi me aprofundar um pouco mais. Achei a sintaxe bem intuitiva e agradável, e já consigo desenvolver boas soluções utilizando Python.

JavaScript

Aprendi JavaScript enquanto estudava HTML e CSS através do canal 'Curso em Vídeo' no YouTube. Tenho conhecimento dos conceitos básicos e já adquiri alguma experiência prática com a linguagem.

HTML

Obviamente, este site foi desenvolvido inteiramente em HTML. Aprendi a linguagem há alguns anos, quando tive o interesse de criar meu primeiro site na web.

CSS

Assim como o HTML, aprendi CSS quando decidi criar meu primeiro site. Ele me permitiu estilizar as páginas e trazer mais vida ao design.

Bootstrap

Aprendi Bootstrap quando fui aprender HTML e CSS. Queria criar meu primeiro site que ficasse bonito e funcional em todos os dispositivos, desde notebooks até celulares. Foi então que descobri este framework para ajudar a alcançar esse objetivo.

Visual Studio Code

Utilizo o Visual Studio Code para programar o front-end e fazer as principais conexões com o back-end.

Git

Uso o Git para manter meus sistemas atualizados no GitHub, evitando a necessidade de fazer atualizações manualmente.

IDE eclipse

Para mim, a melhor IDE para back-end é o Eclipse. Utilizo-o para meus projetos em Java e acho incrível para esse propósito. Também estou pensando em começar a estudar PHP e sei que o Eclipse é uma das melhores opções para essa linguagem.
